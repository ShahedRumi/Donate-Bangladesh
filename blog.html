Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface that represents the structure of an HTML as a tree of objects, allowing for the manipulation of its content and structure.

How do you select an element from the DOM?

You can select an element from the DOM using methods like getElementsByClassName(), document.querySelector() or document.getElementById().

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in which a single event listener is added to a parent element to manage events for multiple child elements, improving efficiency and reducing memory usage.

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes using element.setAttribute() and its styles using element.style.property = value.
